# PersonaForge | OpenClaw 人設產生器

![Version](https://img.shields.io/badge/版本-1.0.0-blue)
![License](https://img.shields.io/badge/授權-MIT-green)

✨ **為你的 OpenClaw AI 打造獨一無二的人格**

選擇多種人設特質，融合成專屬於你的 AI 助理，下載四個設定檔即可使用！

![Preview](https://img.shields.io/badge/預覽-PersonaForge-orange)

---

## 🎯 功能特色

- **25+ 種人設選擇** — 職場、角色扮演、娛樂、生活、特殊興趣
- **跨分類混搭** — 選好幾種也沒問題，AI 會自動融合
- **即時預覽** — 選完立刻看到融合後的人格描述 + 說話範例
- **引導問答** — 填寫 USER.md 所需的基本資料
- **一鍵下載** — 產出 SOUL.md、USER.md、AGENTS.md、MEMORY.md 四個檔案
- **Peter 風格指南** — 內建專屬風格（不罐頭、有觀點、简洁、直言不諱）

---

## 🚀 使用方式

### 方法一：線上使用

直接打開 [GitHub Pages](https://chung223.github.io/persona-forge/) 即可使用！

### 方法二：本地端

```bash
# Clone 下來
git clone https://github.com/chung223/persona-forge.git
cd persona-forge

# 用瀏覽器打開
open index.html
```

---

## 📦 產出檔案

下載後會得到一個 ZIP，解壓縮後包含：

| 檔案 | 用途 |
|------|------|
| `SOUL.md` | AI 人格、說話風格 |
| `USER.md` | 你的基本資料、偏好 |
| `AGENTS.md` | 工作規則、記憶方式 |
| `MEMORY.md` | 長期記憶庫 |

---

## 📂 安裝方式

將下載的四個 `.md` 檔案複製到你的 OpenClaw 工作目錄：

```bash
cp *.md ~/.openclaw/workspace/
```

然後重啟 OpenClaw 就會載入新人格！
